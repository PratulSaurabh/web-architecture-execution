# 1.1.6: HTTP Methods & Status Codes

- **Methods (Verbs):** Define the client's intent. The main ones are `GET` (read), `POST` (create), `PUT`/`PATCH` (update), and `DELETE` (destroy).
- **Status Codes (Outcomes):** A 3-digit code from the server indicating the result.
  - **2xx (Success):** `200 OK`, `201 Created`
  - **3xx (Redirection):** `301 Moved Permanently`
  - **4xx (Client Error):** `400 Bad Request`, `401 Unauthorized`, `403 Forbidden`, `404 Not Found`
  - **5xx (Server Error):** `500 Internal Server Error`
- **RESTful Design:** A well-designed API uses this vocabulary correctly. The URL is the "noun" and the method is the "verb."
